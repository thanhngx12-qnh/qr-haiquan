Bước 1

1. Cấu Hình Thiết Bị KAW-K136G / KAW-K136F

Chúng ta cần nạp phím F12 làm tiền tố gửi trước (Prefix) để hệ điều hành kích hoạt file Excel đối soát tự động khi quét mã.

  • Mở cuốn Sách hướng dẫn sử dụng (User Manual) đi kèm trong hộp máy quét.
  • Tìm đến chương mục Add Prefix (Thêm tiền tố) hoặc Prefix Settings.
  • Cầm máy quét lên và quét lần lượt các mã vạch sau theo đúng thứ tự:
    • Quét mã: Clear All Prefix (Xóa sạch cài đặt cũ tránh xung đột).
    • Quét mã: Add Prefix (Yêu cầu thiết lập thêm ký tự đứng trước).
    • Quét mã: Phím F12 (Xem ở bảng danh sách mã phím chức năng ở trang cuối của tài liệu).
  • Máy quét kêu "Bíp Bíp" báo hiệu lưu thành công.

Cơ chế đầu ra của máy quét

Sau khi quét, mỗi lần quét mã QR, máy sẽ giả lập nhấn nút F12 trước để gọi lệnh từ AutoHotkey, sau đó gõ dữ liệu 8 cột và kết thúc tự động bằng lệnh Enter để xuống dòng.
Bước 2

2. Trình Cấu Hình Script AutoHotkey Tự Động

Nhập từ khóa hoặc tên file Excel đối soát bãi xe của bạn vào ô dưới đây. Mã Script sẽ tự động cập nhật ngay tức thì!

  • Tải và cài đặt phần mềm AutoHotkey v1.1 từ trang chủ autohotkey.com.
  • Tại thư mục làm việc, nhấp chuột phải chọn New -> AutoHotkey Script. Đặt tên là TuDongQuetExcel.ahk.
  • Nhấp chuột phải vào file vừa tạo -> chọn Edit Script. Xóa hết nội dung cũ, dán đoạn mã đã copy ở trên vào và lưu lại (Ctrl + S).
  • Kích đúp chuột để chạy file TuDongQuetExcel.ahk. Xuất hiện biểu tượng chữ H màu xanh lá ở khay hệ thống là hoàn thành!
Bước 3

3. Định Dạng File Excel Nhận Liệu

Để tránh lỗi Excel tự động xóa mất số "0" ở đầu mã số thuế doanh nghiệp, hãy chắc chắn đã thiết lập định dạng cột như sau:

Cột Excel Trường thông tin nhận tương ứng Kiểu định dạng (Format)
Cột A 1. Mã số thuế Text (Văn bản)
Cột B 2. Tên công ty Text (Văn bản)
Cột C 3. Số tờ khai Text (Văn bản)
Cột D 4. Địa điểm tập kết Text (Văn bản)
Cột E 5. Biển kiểm soát Text (Văn bản)
Cột F 6. Số hiệu Container Text (Văn bản)
Cột G 7. Mặt hàng Text (Văn bản)
Cột H 8. Người khai hải quan Text (Văn bản)
  • Bôi đen toàn bộ các cột từ A đến H trong file Excel đối soát bãi xe.
  • Tại thanh menu trang chủ Home -> ô lựa chọn định dạng số (Number Format) đổi từ General thành Text.
  • Bấm chọn sẵn vào ô A2 (ô đầu tiên của cột Mã số thuế) để sẵn sàng đón nhận lượt quét.
Bước 4

4. Vận Hành Hằng Ngày

LƯU Ý BẮT BUỘC TRƯỚC KHI QUÉT MÃ

Nhấn tổ hợp phím Ctrl + Shift để chuyển bộ gõ tiếng Việt (Unikey / EVKey) sang Tiếng Anh (Biểu tượng chữ E màu xanh). Điều này giúp máy quét nhập liệu cực nhanh mà không bị lỗi dính dấu tiếng Việt.
  • Mở sẵn file Excel đối soát (ví dụ: DoiSoat_BaiXe.xlsx) và chọn sẵn ô nhập liệu đầu tiên. Bạn có thể ẩn file xuống dưới Taskbar.
  • Bạn có thể làm việc trên bất kỳ phần mềm chuyên ngành nào khác.
  • Khi phương tiện trình diện mã QR, đưa mã vào tầm quét của thiết bị và ấn quét.
  • Hệ thống tự động nhận tín hiệu, gọi file Excel đối soát nổi lên và điền nhanh 8 trường thông tin thẳng hàng một cách hoàn hảo!
Sự cố

Giải Quyết Các Sự Cố Thường Gặp

1. Bấm quét xong Excel tự động hiện bảng "Save As" (Lưu file dưới dạng...) ▼
Nguyên nhân: Do phần mềm AutoHotkey chưa được khởi chạy ngầm trên Windows nên không "nuốt" được phím F12 gửi từ đầu máy quét.
Khắc phục: Hãy kiểm tra khay hệ thống (góc dưới bên phải màn hình) xem có biểu tượng chữ H màu xanh lá cây không. Nếu không có, hãy kích đúp chuột vào file TuDongQuetExcel.ahk để khởi chạy lại.
2. Chữ gõ ra bị lỗi font dính dấu loạn xạ (Ví dụ: "Anh Đức" gõ thành "Anh d d u u c") ▼
Nguyên nhân: Do Unikey hoặc EVKey trên máy tính của bạn đang để chế độ Tiếng Việt (Chữ V). Vì máy quét nhập liệu cực nhanh, bộ gõ sẽ hiểu nhầm các ký tự liên tiếp là phím bấm dấu.
Khắc phục: Nhấp vào biểu tượng Unikey đổi sang chế độ gõ Tiếng Anh (Chữ E) hoặc nhấn tổ hợp phím Ctrl + Shift trước khi bắt đầu ca trực quét mã.
3. Máy quét kêu "Bíp" nhưng file Excel đối soát không tự động nổi lên ▼
Nguyên nhân: Tên của file Excel bạn đang mở không chứa từ khóa cấu hình trong đoạn mã Script (Ví dụ bạn cấu hình tìm chữ "DoiSoat" nhưng tên file Excel thực tế là "Bao_Cao_Tuan.xlsx").
Khắc phục: Đổi tên file Excel của bạn cho chứa từ khóa đó (Ví dụ: "DoiSoat_Bãi_Xe.xlsx") hoặc chỉnh sửa lại tên cấu hình trong ô nhập của Bước 2 và cập nhật lại mã code vào file AHK.
